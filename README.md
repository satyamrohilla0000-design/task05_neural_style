# Neural Style Transfer

## Overview
This project applies the artistic style of one image to the content of another image using Neural Style Transfer (NST). Deep learning techniques are used to combine content and style representations into a new artistic image.

## Objectives
- Understand Neural Style Transfer.
- Learn feature extraction using Convolutional Neural Networks.
- Transfer artistic styles between images.
- Generate visually appealing artwork.

## Technologies Used
- Python
- TensorFlow / PyTorch
- OpenCV
- NumPy
- Matplotlib

## Working Principle
Neural Style Transfer uses a pre-trained CNN (typically VGG19) to extract:

### Content Features
Represent the structure and objects present in the image.

### Style Features
Represent textures, colors, and artistic patterns.

The final image is generated by minimizing:
- Content Loss
- Style Loss
- Total Loss

## Workflow
1. Load content image.
2. Load style image.
3. Extract features using VGG19.
4. Calculate content and style losses.
5. Optimize generated image.
6. Save the stylized output.

The generated image preserves the original content while adopting the artistic characteristics of the style image.
